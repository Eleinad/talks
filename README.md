## <a name="pycon_2026"></a> PyCon Italia 2026 — Bologna

**Folder:** `pycon_2026/`

**Title:** *"Your Photo Gallery, but Smarter - A Local-First Semantic Image Search System That Runs on Your Laptop"*

This talk presented a fully local image semantic search and clustering tool. The system lets you search a personal photo library using plain-text descriptions or a reference image, leveraging CLIP embeddings and a FAISS index for fast similarity retrieval. It also features an unsupervised clustering pipeline built with BERTopic, UMAP, and HDBSCAN, with human-readable topic labels generated automatically using BLIP and a local LLM via Ollama — no data ever leaves your machine.

## <a name="pycon_2025"></a> PyCon Italia 2025 — Bologna

**Folder:** `pycon_2025/`

**Title:** *"Local LLM fine-tuning: a practical example"*

This session began with a brief introduction to local large language models (LLMs) and fine-tuning. I then demonstrated how to fine-tune an open-weight model like Meta's LLaMA, and finally showcased how to integrate the fine-tuned model into a ChatGPT-style user interface.

If you missed the talk, you can watch it again [here](https://www.youtube.com/live/a8PI4sgWXE8?si=tp7WFzym1tU3kA1J&t=19794)!

## <a name="pydata_roma_capitale_2025"></a> PyData Roma Capitale 2025

**Folder:** `pydata_roma_capitale_2025/`

**Title:** *"Exploring Multi-Agent Systems with AutoGen"*

This session introduced the concept of collaborative AI agents and their applications in solving complex problems. I showcased AutoGen, an open-source framework by Microsoft that simplifies the creation and coordination of multiple agents, and walked through three practical demos: agent-to-agent conversation, a simulated health service onboarding system, and an autonomous Bitcoin market analysis.
